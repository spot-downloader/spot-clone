# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-28

### Added
- Initial release
- Vue 3 + Vite frontend application
- Spotify-like UI dengan dark theme
- Progressive Web App (PWA) support
- Service Worker untuk offline caching
- PWA install functionality
- Navidrome/Subsonic API integration
- Playlist viewing dan management
- Responsive design (mobile, tablet, desktop)
- Tailwind CSS styling
- Docker support dengan Nginx
- Complete documentation (README, CONTRIBUTING, SECURITY)

### Features
- Home page dengan PWA status cards
- Playlist list view
- Individual playlist view
- Music streaming dari Navidrome
- Install as PWA button
- Offline mode support
- MD5 authentication untuk Subsonic API

### Tech Stack
- Vue 3.5
- Vite 7
- Vue Router 4
- Axios
- Tailwind CSS 3
- vite-plugin-pwa
